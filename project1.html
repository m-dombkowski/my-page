Weather app

This weather app was made as one of my portfolio projects. Simple website which allows user to check current temp, humidity and pressure in one of the thousands city that can be chosen using OpenWeather API, as well as possibility to check 12 hours forecast with actual temp, feel temp, wind speed and graph that visualize temperature data

Tech Used

  • HTML
  • CSS
  • Javascript
  • OpenWeather API
  • Chart.js
  • Local Storage
